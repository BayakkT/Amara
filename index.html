Conseils de sécurité

Cette section donne quelques conseils simples pour aider les utilisatrices à se déplacer avec plus de sécurité, surtout dans les zones signalées comme sensibles.

Rester dans les zones éclairées

Le soir, il est préférable de choisir les rues principales et les endroits bien éclairés, même si le trajet est un peu plus long.

Éviter les lieux isolés

Les parcs, parkings ou petites rues vides peuvent être moins sûrs, surtout tard le soir.

Prévenir une personne de confiance

Avant un déplacement tardif, il peut être utile de prévenir un proche ou de partager sa localisation.

Appeler les numéros utiles

En cas de danger, il faut contacter la police au 17. Pour les violences faites aux femmes, le numéro 3919 peut être utilisé.

About us

Amara est un projet de site web imaginé pour aider les femmes à mieux anticiper leurs déplacements en ville. L’idée principale est de proposer une carte simple et lisible, avec des zones classées selon leur niveau de risque.

Le nom Amara est associé à l’idée de femme et de guide. Le site cherche donc à accompagner les utilisatrices en leur donnant des informations utiles, comme les zones à éviter, les lieux à surveiller et les conseils de sécurité.

Le projet repose aussi sur une logique participative : les utilisatrices peuvent consulter les alertes existantes et, plus tard, ajouter leurs propres signalements avec une zone géographique, une rue, une heure et une description courte.
